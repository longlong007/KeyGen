# SecurePass Manager - 安全密码管理器

一个用 Python 编写的本地密码管理工具，支持安全密码生成和加密存储。

## 功能特点

- **安全密码生成**: 使用密码学安全的随机数生成器 (secrets 模块)
- **加密存储**: 使用 Fernet 对称加密，数据本地存储
- **灵活配置**: 自定义密码长度、字符类型
- **密码管理**: 添加、编辑、删除、搜索密码记录
- **一键复制**: 快速复制密码到剪贴板

## 环境要求

- Python 3.8+
- Windows / macOS / Linux

## 安装

1. 克隆或下载本项目
2. 安装依赖:

```bash
pip install -r requirements.txt
```

## 使用方法

1. 启动程序:

```bash
python main.py
```

2. **首次运行**: 点击"首次设置"按钮，设置您的主密码（至少6位）

3. **日常使用**: 输入主密码登录

### 主要功能

- **生成密码**: 点击"生成密码"按钮，可配置密码长度和字符类型
- **添加记录**: 点击"添加记录"手动输入账号密码
- **查看密码**: 双击列表中的记录查看详情，或右键菜单选择"查看密码"
- **编辑/删除**: 右键点击记录进行编辑或搜索**: 使用顶部搜索框过滤删除

## 文件说明

- `main.py` - 主程序
- `requirements.txt` - Python 依赖
- `passwords.dat` - 加密的密码数据文件（首次运行后生成）
- `salt.bin` - 加密盐值文件（首次运行后生成）

## 安全说明

- 主密码用于生成加密密钥，不会以明文形式存储
- 密码数据使用 Fernet (AES) 对称加密
- 每次启动都需要输入正确的主密码才能解密数据
- 请妥善保管您的主密码，忘记后无法恢复数据
